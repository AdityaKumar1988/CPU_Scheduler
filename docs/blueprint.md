# **App Name**: SchedulerSim

## Core Features:

- Task Input: Input form for tasks including Task ID, Arrival Time, Burst Time, Priority, and Quantum.
- Task Table: Display tasks in a table format.
- Algorithm Selection: Selection of scheduling algorithms including FCFS, SJF (preemptive & non-preemptive), Priority (preemptive & non-preemptive), Round Robin, EDF, and RMS.
- Gantt Chart Visualization: Visualize scheduled tasks using a Gantt chart.
- Results Table: Display task completion time, waiting time, turnaround time, response time and deadline misses (if RMS/EDF used).
- Statistics Output: Output overall statistics like CPU utilization, average waiting time and turnaround time, and total deadline misses.
- Export to CSV: Enable the simulator's results to be exported to a CSV file.

## Style Guidelines:

- Primary color: Deep blue (#3F51B5) to represent a stable and efficient system.
- Background color: Light gray (#F5F5F5) to ensure readability and reduce eye strain.
- Accent color: Teal (#009688) for highlighting interactive elements and important information.
- Body and headline font: 'Inter' (sans-serif) for clear, modern readability.
- Use simple, consistent icons from a library like Material Icons to represent actions and data.
- Responsive layout using CSS Grid and Flexbox for clear arrangement on various screen sizes.
- Use subtle transitions and animations to enhance user experience.
